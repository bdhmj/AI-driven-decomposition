# AI-driven Project Estimation

Рабочее пространство для оценки и декомпозиции IT-проектов. Работает через Claude Code (VS Code extension) — без API, на подписке.

## Что делает

Берёт описание проекта от клиента и за три этапа выдаёт:
1. **Техническое задание** (.docx) — трёхчастный документ: MVP-скоуп с границами, открытые вопросы, чеклист расширений
2. **Декомпозицию** (.xlsx) — задачи по модулям с оценками в днях, разделение MVP / Post-MVP
3. **GANTT-диаграмму** — визуальное расписание по специалистам с учётом рабочих дней

## Быстрый старт

### 1. Установка

```bash
git clone git@github.com:np-prsnl/AI-driven-decomposition.git
cd AI-driven-decomposition
pip install -r requirements.txt
```

### 2. Открой в VS Code с Claude Code

Убедись, что расширение Claude Code установлено и активно.

### 3. Положи описание проекта в `input/`

Любой формат: `.md`, `.txt`, `.pdf`, `.docx` — или несколько файлов сразу.

### 4. Запусти оценку

Скажи Claude Code:
```
Оцени проект из input/
```

Claude пройдёт три этапа:
- Проверит полноту описания (задаст вопросы если нужно)
- Сгенерирует ТЗ → `output/spec.md` + `output/Техническое_задание.docx`
- Декомпозирует → `output/decomposition.json` + `output/Оценка_проекта.xlsx`

### 5. Заархивируй результаты

```bash
./scripts/archive.sh "Название проекта"
```

Всё из `input/` и `output/` переедет в `projects/YYYY-MM-DD_Название/`.

## Структура ТЗ

| Часть | Что содержит | Для кого |
|-------|-------------|----------|
| **MVP — Базовый скоуп** | Функциональные требования по модулям с явными границами «делаем / не делаем» | Контракт с клиентом + требования для разработки |
| **Открытые вопросы** | Неоднозначности с вариантами ответов и влиянием на оценку | Обсуждение с клиентом до старта |
| **Чеклист расширений** | Таблица всех возможных усложнений с оценкой в днях и рекомендацией | Меню опций — клиент выбирает, видя стоимость |

## Структура репозитория

```
input/              ← описания проектов от клиентов
output/             ← результаты текущего проекта
projects/           ← архив завершённых проектов
skills/             ← скиллы Claude Code
  analyze-request/  ← анализ полноты запроса
  generate-spec/    ← генерация трёхчастного ТЗ
  decompose-tasks/  ← декомпозиция на задачи
scripts/            ← утилиты
  build_docx.py     ← markdown → .docx
  build_xlsx.py     ← JSON → .xlsx с GANTT
  archive.sh        ← архивация проекта
references/         ← калибровочные примеры
```
